# Ames-Housing-Price-Prediction

Building a machine learning web app to predict house prices in Ames, Iowa.

The prediction model running on the app ranks in the top 8% of Kaggle's [House Price Prediction Competition leaderboard](https://www.kaggle.com/c/house-prices-advanced-regression-techniques/overview) and top 1% of [Housing Prices Competition for Kaggle Learn Users leaderboard](https://www.kaggle.com/c/home-data-for-ml-course/overview) (as of 10/25/2021).

Kaggle notebook outlining the model building process is available [here](https://www.kaggle.com/ruthgn/house-prices-top-8-featengineering-xgb-optuna/notebook).
